Rock Paper Scissors Project

What it is: This is a simple interactive game that recreates Rock Paper Scissors in the browser. It is designed to practice basic game logic, random computer choices, and score tracking while keeping the interface easy to follow.

How to use it: Click Rock, Paper, or Scissors to make your move. The game immediately shows the result, updates the score, and lets you keep playing. The reset button clears the current round history so you can start fresh whenever you want.

What I learned: This project helped me practice condition handling and feedback design. I learned how to compare player choices against computer choices, show the result clearly, and keep the game feeling responsive instead of static.
