Typing Speed Test Project

What it is: This typing test is a browser-based practice tool built to measure words per minute, accuracy, and consistency. It is meant to help students practice typing in a structured way instead of treating speed as the only goal.

How to use it: Start the test, type the displayed text into the input area, and watch the live stats update as you go. You can change the duration, reset the round, and use the results to see whether you are improving in speed, accuracy, or both.

What I learned: Building this project taught me how to work with timers, input events, and live UI updates. I also learned that a good practice tool needs clear feedback, because users improve faster when they can see exactly what changed.
